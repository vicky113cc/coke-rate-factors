# coke-rate-factors

台塑－影響高爐焦比之
關鍵因素分析


在探討高爐煉鐵製程中影響焦比變異的關鍵因素，並運用機器學習方法分析實際生產數據，以協助產線達成能源效率提升與成本優化。

本研究以台塑某高爐為實例，結合資料整合、特徵工程與多種預測模型，系統性識別影響焦比的重要製程參數。
進一步導入 SHAP（SHapley Additive Explanations）解釋型人工智慧技術進行特徵重要性排序，並運用 GRF（Generalized Random Forest）方法計算平均處置效果（ATE），精確量化各變數
對焦比的實質影響，提出可行的製程參數調控建議。

關鍵字：焦比、機器學習、SHAP、GRF、特徵重要性、資料分析、高爐製程、製程優化、能源效率
